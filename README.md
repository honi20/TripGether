# ✈️ TripGether

**TripGether**는 여행의 계획부터 매칭, 추억 저장까지 여행의 전 과정을 기록할 수 있는 서비스입니다.

<br>

## 📱 서비스 화면

### 📝 여행 계획

- 여행 계획을 세울 수 있습니다.
- 날짜 선택 시 일자 별 탭이 생성되어 일자별 계획이 가능합니다.
- 해당 일자에 계획된 장소 정보가 지도의 핀으로 표시됩니다.
- 장소 검색 시 장소 정보, 브이로그, 주변 관광지 정보가 제공됩니다.
